# Android Group Messenger Application

A real-time group chat app for Android. Users can create/join groups, send messages, view chat history, and manage profiles. Built with Java/Kotlin and Firebase for instant messaging and notifications.

## Features
- User Authentication (Sign up / Log in)
- Create and join groups
- Real-time messaging
- Chat history storage
- User profiles
- Push notifications for messages

## Technologies Used
- Language: Java / Kotlin
- Database: Firebase Realtime Database / Firestore
- UI: XML layouts with Material Design
- Networking: Firebase Cloud Messaging / Socket Programming
